Employee Management System
A Java program to manage employee data with sorting and searching features.

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

What It Does

-Load employee data from files
-Sort employees by name (Quick Sort)
-Search for employees (Binary Search)
-Add new employees
-Show company structure (Binary Tree)
-Clean console menu

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

How to Run

bash
javac \*.java
java Main

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Main Features

Load Data - Read employee information from file
Sort - Arrange employees by last name
Search - Find employees quickly
Add New - Create new employee records
Show Tree - Display company structure
View All - See all employees
Exit - Close program

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Files Needed

Main.java - Main menu
DepartmentStore.java - Main logic

Department.java - Department data

EmployeeBinaryTree.java - implements binary tree with level-order insertion 
Employee.java - Employee data

TreeNode.java - Tree container

Manager.java - Positions Validation

Applicants\_Form.txt - Your employee data
Just run App.java and follow the menu!
